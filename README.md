# Zerodha Clone

A responsive front-end clone of the Zerodha website built using HTML, CSS, and JavaScript. This project was created to practice front-end web development and improve UI design skills.

## 🚀 Features

- Responsive homepage layout
- Clean and modern user interface
- Navigation bar
- Hero section
- Interactive elements using JavaScript
- Mobile-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Zerodha-Clone/
│── index.html
│── style.css
│── script.js
```

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/sindhu-p005/Zerodha-Clone-.git
   ```

2. Open the project folder.

3. Open `index.html` in your browser.
